# 📊 Projeto Django

Aplicação web desenvolvida com **Django** para gerenciamento e desenvolvimento de funcionalidades web utilizando boas práticas de desenvolvimento e organização de código.

## 🚀 Tecnologias utilizadas

- Python
- Django
- SQLite (padrão do Django)
- python-dotenv
- SQLParse

Dependências principais:

- Django 6
- python-dotenv
- asgiref
- tzdata
